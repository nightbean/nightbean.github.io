HEXO+GitHub搭建个人博客——安装部署

创建github代码库

  • 注册GitHub帐号,如果已有，则省略。
  • 登录帐号，点击start a project 或 new repository 创建一个代码库，代码库名称为 yourname.github.io 即博客访问地址。
  • 创建代码库需要验证邮箱，按照提示验证即可。

打开浏览器，localhost:4000 ，就可以看到本地的个人博客。

配置SSH密钥

  • 设置Git身份信息
  • 生成SSH密钥

查看是否已有SSH密钥

如果没有则提示： No such file or directory
如果有则进入~/.ssh路径下（ls查看当前路径文件，rm * 删除所有文件）

生成密钥

复制密钥

浏览器登录自己的github页面, 进入”Account Settings”, 再点击左边的”SSH Key”可以看到自己上传过的SSH公钥列表. 再点击”Add SSH Key”新增一个公钥, 直接粘贴即可(上一步已经使用xclip把公钥id_rsa.pub复制到剪切板了)；
